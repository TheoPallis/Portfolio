Case Law Analysis

Analysis of 13 years of case law

Project description

The client requested the analysis of 340 law cases, regarding the nature of the transgressions and the imposed fines.

Web Scraping

The client did not provide any dataset, so Python was used in order to gather the necessary data from the web.

  • Python Code

Cleaning the data

Given the relatively small size of the dataset, Excel was chosen in order to :

  1. Transpose, merge, split, filter and clean the data.
  2. Remove blank entries .
  3. Apply Conditional Formatting.

    • Finished Sheet

Dashboard

A Dashboard was created, per the request of the client, displaying the following information per market segment:

  • Total and indivual number of transgressions.
  • Total and indiviudal number of imposed fines.
  • Total and indivual number of decisions approving of the business collaborations.
  • Number of cases where the result of an appeal was the reduction of the imposed fine.
  • Dashboard
