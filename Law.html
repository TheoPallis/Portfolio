Case Law Analysis

Analysis of 340 law cases

Project description

The client requested the analysis of 340 law cases, regarding the nature of the transgressions and the imposed fines.

Web Scraping

The client did not provide any dataset, so Python was used in order to gather the necessary data from the web.

(Data source : https://www.epant.gr/apofaseis-gnomodotiseis.html)

  • Python Code

Cleaning the data

Given the relatively small size of the dataset, Google Sheets was chosen in order to :

  1. Transpose and merge the data.
  2. Clean cell contents.
  3. Split the data into relevant columns.
  4. Remove blank entries.
  5. Apply Conditional Formatting to facilitate the previewing of the dataset.
  6. Filter the data based on the :
    • Nature of the trangression
    • Relevant market
    • Imposed fine
    • Appeal application
    • Appeal result
  7. Create the relevant documentation and timeline of the project.

    • Finished Sheet

Dashboard

A Dashboard was created, per the request of the client, displaying the:

  • Total number of transgressions per market segment.
  • Number of transgressions per market segment.
  • Total number of imposed fines.
  • Number of imposed fines per market segment.
  • Total number of decisions that approve the business unions.
  • Number of decisions that approve the business unions per market segment.
  • Number of cases where the result of an appeal was the reduction of the imposed fine.
  • Dashboard

Summary

Overall, this was a very exciting and relatively demanding project, as an uncanny level of attention to detail was required.
I was also given the opportunity to apply my legal knowledge.
The client was extremely satisfied with the result
(given that I noted some law cases of further interest during my analysis).
