E-commerce Analysis

Uncovering customer insights

Project description

This project represents another typical analysis of sales data using Pandas this time.

The purpose of the project was to uncover insights, via analyzing the collected data of an e-shop.

Cleaning the data

  1. Merged all files into a single file.
  2. Removed rows with NaN entries.
  3. Removed irrelevant text in all entries.
  4. Converted column type to appropriate type.
  5. Added relevant attribute columns.
  • JUPYTER NOTEBOOK

KPIs

  • Best month for sales,
  • Profit earned during that month,
  • Best selling location.
  • Best time to display advertisements.
  • Products most often sold together.
  • Best selling products.
