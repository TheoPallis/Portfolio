Covid Analysis

Analysis of the Pandemic

Project description

This project takes a look into the particular death statistics of the pandemic, in an attempt to reach sensible conclusions.

This project focuses on the state of the pandemic in Greece during May-November 2021.

Cleaning the data

  1. Replaced nulls,blanks with 0s.
  2. Created temporary table containing only the Greek records.
  3. Created temporary table containing only the Greek records for the last six months.
  • Sql code

Analysing the data

  1. Number of total deaths compared to the number of new deaths.
  2. Average and highest number of new deaths per day.
  3. Number of deaths per day on October 2021.
  4. Current vaccinated percentage of the population.
