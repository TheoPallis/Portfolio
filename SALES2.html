Sales Analysis

Sales Analysis by item type

Project description

This project represents a typical analysis of sales data.

Sample Dataset used : https://eforexcel.com/wp/downloads-18-sample-csv-files-data-sets-for-testing-sales (5M rows)

Cleaning the data

  1. Checking for typos, nulls, blanks, duplicates.
  2. Checking for outliers.
  • SQL code

KPIs

  1. Most recent and oldest 5 item orders.
  2. Most sold and most profitable item types.
  3. Total profit per location for each and all individual item types.
  4. Locations with the highest number of sales for each item type.
  5. Annual revenue for each item type.
  6. Most profitable month for each item type.
  7. Sales per day for each item type.
  8. Item type with the most sales for each year.

  • SQL Code

Dashboard

Created a fully interactive dashboard displaying, based on the filters applied :

  1. The most profitable item type and distribution channel.
  2. The total revenue generated for the most profitable item type.
  3. The total units sold for each item type, broken further down by distribution channel.
  4. The most profitable months for an item type.
  5. The total profit for each item type, broken further down by distribution channel.
